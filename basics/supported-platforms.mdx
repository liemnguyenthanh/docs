# Supported Platforms

Stratum operates across multiple blockchain networks, providing users with access to diverse DeFi ecosystems and investment opportunities.

## Primary Networks

### Ethereum

<Card title="Ethereum Mainnet" icon="ethereum">
- **Network**: Ethereum
- **Symbol**: ETH
- **Features**: Full DeFi ecosystem access
- **Gas Token**: ETH
- **Status**: Fully supported
</Card>

<Columns cols={2}>
<Card title="Ethereum Benefits" icon="check">
- Largest DeFi ecosystem
- Most liquidity available
- Extensive protocol support
- Mature infrastructure
</Card>

<Card title="Ethereum Considerations" icon="exclamation">
- Higher gas fees
- Network congestion
- Slower transaction times
- Higher costs for small transactions
</Card>
</Columns>

### Polygon

<Card title="Polygon Network" icon="polygon">
- **Network**: Polygon (MATIC)
- **Symbol**: MATIC
- **Features**: Low-cost transactions
- **Gas Token**: MATIC
- **Status**: Fully supported
</Card>

<Columns cols={2}>
<Card title="Polygon Benefits" icon="check">
- Very low gas fees
- Fast transaction times
- Ethereum compatibility
- Growing DeFi ecosystem
</Card>

<Card title="Polygon Considerations" icon="exclamation">
- Smaller ecosystem than Ethereum
- Bridge requirements for some assets
- Less liquidity for some pairs
- Newer network with less history
</Card>
</Columns>

### Binance Smart Chain

<Card title="BSC Network" icon="binance">
- **Network**: Binance Smart Chain
- **Symbol**: BNB
- **Features**: Low fees, high performance
- **Gas Token**: BNB
- **Status**: Fully supported
</Card>

<Columns cols={2}>
<Card title="BSC Benefits" icon="check">

- Fast block times
- Large user base
- Binance ecosystem integration
</Card>

<Card title="BSC Considerations" icon="exclamation">

- Smaller DeFi ecosystem
</Card>
</Columns>
